Praktek Pemograman Web

Sesi 1

Praktek Pemograman Web akan membahas tentang cara menampilakn halaman web dengan menggunakan HTML, CSS
dan JavaScript. Mahasiswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan
pemahaman tersebut kemudian menuliskan hasil kreasinya sendiri.

CSS

Cascading Style Sheet merupakan standar untuk mengatur tampilan yang ada pada suatu berkas HTML. CSS bukan
bahasa Pemograman tetapi merupakan bahasa deklaratif yang dinyatakan langsung untuk menyatakan suatu tampilan, tidak
terdapat logika, percabangan, atau perulangan.

JavaScript

JavaScript merupakan bahasa pemograman yang berjalan disisi browser, disisi pengguna, atau di peramban bukan di server
